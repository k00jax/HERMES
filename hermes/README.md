# Hermes Monorepo

Scaffolding for firmware, Linux (Odroid), hardware docs, and tooling.

## Firmware notes

- nRF52840 firmware must be built on PC.
- UF2 is generated via post-build script.
- Do not attempt to build this project on Odroid.
